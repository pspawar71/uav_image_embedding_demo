# uav_image_embedding_demo
# UAV Image Embeddings with ResNet‑18

This repository provides an interactive tutorial demonstrating how to extract
**image embeddings** from UAV RGB imagery using an ImageNet‑pretrained ResNet‑18.

The tutorial illustrates:
- Feature map evolution across ResNet‑18 layers
- Global Average Pooling (GAP)
- Construction of a 512‑dimensional image embedding
- Visualization tools for interpretability
